Project Overview

Background: The primary goal of this group project was to introduce us to the fundamentals of IoT development through hands-on experience with physical microcontrollers, external sensors, actuators, software, and communication protocols.

Key Requirements:

  • Install the necessary Python libraries to interact with the DHT11 sensor
  • Create a Python script that reads and displays the data from the sensor with a timestamp
  • Log the data to a file (e.g. CSV) for further analysis

    • Solution: Integrate the DHT11 sensor with a Raspberry Pi and develop a user-friendly interface that displays the current temperature and humidity readings, alongside their corresponding timestamps. Furthermore, the data must be logged into a CSV file.

    Key Features

  • Real-Time Dashboard and Live Graph:
    • Displays current temperature, humidity and time
    • Displays data points in real time on a line chart
  • Data Logging and Exporting:
    • Users can download the CSV file that contains the readings and timestamps
    • Users can download the graph as an image

    Project Reflection

    This was my first hands-on experience working with hardware, and while the wiring itself was straightforward, I'll admit I was nervous at first. The thought of something exploding, especially since the equipment wasn't mine added real pressure to the situation. The constant fear of accidentally connecting the wrong pin or pulling out the wrong wire was scary

    Despite these anxieties, the project went smoothly for someone working with hardware for the first time. It opened my eyes to how things work on a physical level in relation to software, satisfying my curious nature. Getting to see the data display in real-time on the dashboard was incredibly rewarding...watching the system come to life making the entire effort worthwhile.

    This experience was made enjoyable by my team member and the support from other groups. I'm grateful to my lecturer for expanding my skill set beyond software development, making me more prepared and well-rounded for real-world challenges in the world of work.
